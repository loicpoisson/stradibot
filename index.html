A robot

the violin.

Stradibot is a 7-DOF Flexiv robotic arm performing live violin music using operational space control, force-controlled bowing, solenoid fingering, and real-time MIDI playback — all coordinated by a finite state machine at 1 kHz.

How it works

Six subsystems work in concert to produce music from a standard MIDI file.

♩
MIDI Parsing

A custom C++ parser reads standard .mid files, builds a tempo map, maps each note to a violin string (G3–E5) and fret (0–8), and converts MIDI ticks to real-time seconds.

C++ · MIDI
⚡
Solenoid Fingering

8 solenoid bars press all 4 strings simultaneously at the correct fret. Serial commands are sent over USB at 115 200 baud to an Arduino that drives the solenoids in real time.

Arduino · Serial
↔
Force-Controlled Bowing

The end-effector moves along the bow axis while a force controller maintains a set contact normal to the string. Bow speed and force are tunable at runtime via Redis keys.

SAI2 · Eigen · Redis
⬆
String Transitions

Adjacent strings use a direct lateral move. Non-adjacent changes trigger a lift → lateral → lower trajectory, avoiding collisions with intervening strings during playback.

Motion Planning · FSM
⟳
Finite State Machine

A 9-state FSM (CALIBRATION → LOAD_MIDI → IDLE → SET_FRET → MOVE_TO_STRING → BOW_NOTE → NEXT_NOTE → WAIT_SILENCE → END) coordinates all subsystems with safety timeouts.

C++ · Control
◎
Bow Contact Estimation

A Kalman filter (Python) estimates the bow–string contact state in real time from force sensor readings, enabling adaptive force control during sustained notes.

Python · Kalman Filter

System pipeline

From MIDI file to sound, every step runs in the same 1 kHz control loop.

01
MIDI file

Standard .mid parsed note-by-note with full tempo map and VLQ tick resolution

02
FSM dispatcher

9-state machine coordinates calibration, fret setting, string moves and bowing

03
Redis bus

Real-time shared state: joint angles, bow force & speed, keyboard input

04
SAI2 tasks

MotionForceTask + JointTask compute torques via operational space control

05
Flexiv arm

7-DOF robot executes bowing trajectory with force feedback at the string

06
Arduino

Solenoid driver selects fret 0–8 in sync with each incoming MIDI note event
